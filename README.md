# FilmLog-App
FilmLog is a movie tracking application that allows users to search for movies, track watched movies and view viewing statistics through an interactive dashboard.

## Features
- User registration and login
- Search for movies using an external API
- Add movies to watchlist
- Remove movies from watchlist
- Mark movies as watched
- Reset bumber of times watched
- Track watched movies
- View movie statistics through charts and dashboards
- JWT-based authentication and authorization

## Technologies Used
### Frontend
- Angular
- Ionic
- TypeScript

### Backend
- ASP.NET Core Web API
- C#
- Entity Framework Core

### Database
- SQL Server

## Project Structure
- Frontend: Angular/Ionic application
- Backend: ASP.NET Core Web API
- Database: SQL Server

## How to Run the Project

### Backend
1. Open the backend solution in Visual Studio.
2. Configure the SQL Server connection string in `appsettings.json`.
3. Run the database migrations.
4. Start the API.

### Frontend
1. Open the frontend project in VS Code.
2. Install dependencies:
```bash
npm install
3. Run using: ionic serve
